# Nabava bikova buse

Radni repozitorij za organizaciju ciljane nabave muske junadi pasmine busa iz linija bikova LB7 i LB10.

## Svrha

Cilj je pripremiti dokumentaciju, upite i evidencije potrebne za pronalazak gospodarstava s rijetkim linijama LB7 i LB10, provjeru rodovnika i genetike te moguce kasnije licenciranje kvalitetnih grla kao rasplodnih bikova.

## Sadrzaj

- `01_plan_nabave_i_licenciranja.md` - operativni plan nabave, provjera i licenciranja.
- `02_upit_dobavljacu_i_udruzi.md` - predlosci upita prema Udruzi uzgajivaca buse i prodavateljima.
- `03_evidencija_kandidata.csv` - tablica za pracenje potencijalnih kandidata.
- `04_checklista_prije_kupnje.md` - checklista dokumentacije, zdravlja, premjestaja i kupoprodaje.
- `AGENTS.md` - lokalne upute za rad u ovom workspaceu.

## Napomena

Dokumentacija je radni materijal i prije stvarne kupnje treba provjeriti aktualne podatke s Udrugom uzgajivaca buse, HAPIH-om i nadleznim veterinarskim/registracijskim sluzbama.
