# Checklista prije kupnje kandidata LB7/LB10

## Dokumentacija

- [ ] Broj usne markice je dostavljen
- [ ] Pasmina je potvrdjena kao busa
- [ ] Linija je potvrdjena kao LB7 ili LB10
- [ ] Dostavljeni su podaci o ocu
- [ ] Dostavljeni su podaci o majci
- [ ] Poznat je majcinski rod ili je zatrazen od Udruge/HAPIH-a
- [ ] Provjeren je status u Maticnoj knjizi
- [ ] Dostupan je zootehnicki certifikat ili potvrda da se moze izdati
- [ ] Udruga/HAPIH je potvrdio da grlo ima smisla kao kandidat za licenciranje

## Zdravlje i premjestaj

- [ ] Dostavljena je veterinarska dokumentacija
- [ ] Nema poznatih zapreka za premjestaj
- [ ] Dogovoren je pregled prije utovara
- [ ] Dogovorena je karantena po dolasku
- [ ] Provjeren je status objekta i korisnika objekta za drzane kopnene zivotinje

## Ocjena grla

- [ ] Pregledana je vanjstina
- [ ] Pregledane su noge i papci
- [ ] Pregledana je opca kondicija
- [ ] Pregledan je temperament
- [ ] Snimljene su fotografije i video
- [ ] Usporedjena je usna markica s dokumentacijom

## Kupoprodaja

- [ ] Dogovorena je cijena
- [ ] Dogovoreno je tko placa prijevoz
- [ ] Pripremljen je kupoprodajni ugovor
- [ ] U ugovor je upisana linija LB7/LB10
- [ ] U ugovor je upisan popis dokumentacije koja se predaje kupcu
- [ ] Dokumentacija je preuzeta prije ili najkasnije pri preuzimanju grla
